یک پیشنهاد فوری برای [اوبونتو][ubuntu]:

یک برنامه مدیریت و مرور فایلها و شاخه ها که خیلی ساده و کاربردی بوده به نام Marlin می باشد. البته مرورگر طور پیش فرض در دسکتاپ گنوم، برنامه Nautilus است. این دو برنامه بسیار شبیه به هم هستند.

طریقه نصب آن بر روی اوبونتو:

    sudo add-apt-repository ppa:marlin-devs/marlin-daily
    sudo apt-get update
    sudo apt-get install marlin
